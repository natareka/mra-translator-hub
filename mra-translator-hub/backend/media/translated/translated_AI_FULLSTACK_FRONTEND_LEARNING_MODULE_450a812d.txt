ウェブの基本
1. Web アプリケーションとは何ですか?
定義:
Web アプリケーションは、Web サーバー上で実行され、Web サーバー経由でアクセスされるソフトウェアです。
インターネットを利用したウェブブラウザ。
簡単な説明:
コンピューターやモバイルにソフトウェアをインストールする代わりに、Web サイトを開いて、
ソフトウェアをブラウザで直接使用します。
リアルタイムの例:
 Gmail → ブラウザでメールを送受信
 Amazon → ショッピングプラットフォーム
 Google フォーム → オンライン フォーム送信
 Swiggy/Zomato → 食品注文アプリ
 インターネット バンキング ポータル → 資金移動、残高確認
実生活のたとえ:
モバイルアプリ = インストールされたソフトウェア
Web アプリ = ブラウザー (Chrome、Edge、Safari) 経由で使用します
2. クライアント – サーバー アーキテクチャ
定義:
モデルでは次のことが行われます。
 クライアント = ユーザー デバイス (ブラウザ/モバイル)
 サーバー = データ、ロジック、ファイルを保存するマシン
フロー:
ユーザー（ブラウザ）→リクエスト→サーバー

1. CSS – はじめに
CSSとは何ですか?
CSS (Cascading Style Sheets) は、デザイン、スタイル、レイアウトに使用される言語です
HTML コンテンツ。
HTMLは構造を作成します
CSSで見た目を作る
JavaScript が動作を作成する
CSS を使用しない場合:
 Web サイトはプレーンテキストのように見えます
 色なし
 レイアウトなし
 間隔なし
 デザインなし
 UI/UX なし
現実世界の意味:
HTML = スケルトン
CSS = スキン + 衣服
JS=ビーレイン
リアルタイムの例:
銀行のウェブサイト
 HTML → アカウント詳細構造
 CSS → プロフェッショナルな外観、色、レイアウト
 JS → インタラクション、検証

DOM (Document Object Model) は HTML のツリー構造表現です
ブラウザによって作成されるため、JavaScript は次のことを行うことができます。
 HTMLを読む
 HTMLの変更
 コンテンツの更新
 スタイルを変更する
 イベントを処理する
 要素の作成
 要素を削除する
DOM = HTML と JavaScript の間の橋渡し
簡単な意味:
HTML = 静的ページ
DOM + JS = 動的ページ
