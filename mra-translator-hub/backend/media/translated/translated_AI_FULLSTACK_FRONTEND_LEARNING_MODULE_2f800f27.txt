1. 웹 애플리케이션이란 무엇입니까?
정의:
웹 애플리케이션은 웹 서버에서 실행되고 다음을 통해 액세스되는 소프트웨어입니다.
인터넷을 사용하는 웹 브라우저.
간단한 설명:
컴퓨터/모바일에 소프트웨어를 설치하는 대신 웹사이트를 열고
브라우저에서 직접 소프트웨어를 사용하십시오.
실시간 예시:
 Gmail → 브라우저에서 이메일 보내기/받기
 Amazon → 쇼핑 플랫폼
 Google Forms → 온라인 양식 제출
 Swiggy/Zomato → 음식 주문 앱
 인터넷뱅킹 포털 → 자금이체, 잔고조회
실제 생활 비유:
모바일 앱 = 설치된 소프트웨어
Web App = 브라우저(Chrome, Edge, Safari)를 통해 사용

1. CSS – 소개
What is CSS?
CSS(Cascading Style Sheets)는 디자인, 스타일 및 레이아웃에 사용되는 언어입니다.
HTML 콘텐츠.
HTML은 구조를 생성합니다
CSS가 모양을 만듭니다
JavaScript는 동작을 생성합니다.
CSS가 없는 경우:
 웹사이트는 일반 텍스트처럼 보입니다.
 색상 없음
 레이아웃 없음
 간격 없음
 디자인 없음
 UI/UX 없음
실제 의미:
HTML = 뼈대
CSS = 피부 + 옷
JS = 비
실시간 예:
은행 웹사이트
 HTML → 계정 세부정보 구조
 CSS → 전문적인 모양, 색상, 레이아웃
 JS → 상호 작용, 검증

정의:

 양식 정렬

 모바일 레이아웃
